Minecraft XiaPi server 0.8.1在游戏内添加服务器教程

Created2024-08-19|Updated2024-09-16
|Post Views:

本文章来教会你如何在Minecraft XiaPi server 0.8.1在游戏内添加服务器

先打开游戏点击我箭头所指的位置

再点击我下面剪头所指位置来修改名称

这个名字修改成啥可以看我的这篇文章

修改完成后点击左上角的back来退回到主界面!

然后点击中间的按钮”play”

进入到新页面后点击右上角的”Edit”

随后再点击”External”

进入到新的界面后请将图片中的信息填写到对应的地方

填写好后再点击”Add Server”

然后回到选择页面直接点击名字为”XiaPi”的即可进入服务器
